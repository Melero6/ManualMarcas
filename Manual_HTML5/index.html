Manual HTML5

¿Que es HTML5?

HTML5 es un lenguaje markup (de hecho, las siglas de HTML significan Hyper Text Markup Language) usado para estructurar y presentar el contenido para la web. Es uno de los aspectos fundamentales para el funcionamiento de los sitios, pero no es el primero. Es de hecho la quinta revisión del estándar que fue creado en 1990. La W3C recomendó el uso de HTML5 para transformarse en el estándar a ser usado en el desarrollo de proyectos venideros.


¿De qué se compone HTML5?

El HTML está compuesto por etiquetas, que contienen el elemento y también puede contener atributos. Por ejemplo: 

<html lang="es"></html>

La etiqueta está formada por el elemento <html> y tiene el atributo lang="" (define el idioma del documento), también tiene una etiqueta de cierre que contendrá un "/" delante del elemento. También existen etiquetas con elementos vacios en las que no se pone etiqueta de cierre, como es el caso de la etiqueta <hr>.


Estructura de un documento HTML

La estructura principal de un documento HTML está formado por: 

<!DOCTYPE html>
<html>
	<head>
	</head>
	<body>
	</body>
</html>

En la etiqueta <!DOCTYPE> escribimos el tipo de documento, en este caso html. A continuación abrimos la etiqueta <html></html> que contendrá todo el código.
Dentro de <html></html> tenemos la etiqueta <head></head>, que contiene metadatos de nuestro documento y la etiqueta <body></body>, que contiene toda la parte visual de nuestro documento.


Etiquetas para head

El head contiene metadatos, necesarios para el documento HTML.
Entre el contenido del head podemos destacar las siguientes etiquetas:

  • <title></title>: Asigna un titulo a nuestra página web, que veremos en la pestaña del navegador.
  • <meta>: Sirve para crear metadatos.
    • <metal name="author" content="Jorge">: Creamos un autor y le asignamos un contenido.
    • <metal charset="utf-8">: Asigna una codificación UTF-8 a nuestro documento.

Elementos semánticos estructurales

Estas etiquetas añaden valor semántico al contenido que exista entre ellas. Varias de estas etiquetas se añaden en la versión 5 de HTML.
Etiquetas:

  • <header></header>: Se usa para marcar un grupo de elementos de introducción o de navegación dentro de una sección o documento. Normalmente se usa para incluir los encabezados.
  • <nav></nav>: Se usa para marcar una sección del documento cuya función es la navegación por la página web. Suele estar dentro del header.
  • <aside></aside>: Se usa para marcar un trozo de contenido que está relacionado con el contenido de la página web, pero que no es parte del mismo. Ejemplos de uso serían: glosario de términos, grupos de enlaces a páginas relacionadas, barras laterales.
  • <article></article>: Se usa para marcar contenido independiente que tendría sentido fuera del contexto de la página actual y que podría sindicarse.
  • <section></section>: Se usa para marcar una sección genérica de un documento o aplicación. Una sección, en este contexto, es una agrupación temática del contenido, típicamente con un encabezado.
  • <footer></footer>: Se usa para marcar el pie de una sección o documento y que contiene información sobre el mismo como el autor, licencia, términos de uso.

Títulos

Los títulos de asignan con la etiqueta <hn></hn> siendo n el numero que asigna la importancia y tamaño del título, este número tiene que estar entre el 1 y el 6 (incluidos). Por ejemplo:

<h1></h1>

<h2></h2>

<h3></h3>

<h4></h4>

<h5></h5>
<h6></h6>

Elementos de agrupación

Estas etiquetas nos permiten darle un formato al texto que se muestra en la página web. Las principales son:

  • <p></p>: Define una parte que debe mostrarse como un párrafo.
  • <pre></pre>: El texto aparecen en pantalla con el mismo formato que le asignamos en el documento y con un tipo de letra monoespaciada.
  • <hr>: Añade una línea horizontal, representa un quiebre temático entre párrafos de una sección o articulo o cualquier contenido.

Además, existen etiquetas para formar 3 tipos de listas:

  • <li></li>: Sirve para crear un elemento dentro de las listas ordenas y desordenadas.
  • <ul></ul>: Se usa para crear listas desordenadas.
  • <ol></ol>: Se usa para crear listas ordenandos.
  • <dl></dl>: Se usa para crear una lista de definiciones.
    • <dt></dt>: Se usa para asignar el termino que queremos definir.
    • <dd></dd>: Se usa para añadir una definición al término.

Las listas ordenadas disponen de una serie de atributos:

  • reversed="": Invierte la numeración.
  • type="": Sirve para seleccionar el carácter que queremos usar para numerar la lista.
  • value="": Asigna el numero en que comenzara la lista.

Elementos semánticos de texto

Estas etiquetas aportan un valor semántico al texto que visualizamos. Las principales etiquetas se usan para resaltar el texto:

  • <i></i>: Bastardilla
  • <b></b>: Negrita
  • <u></u>: Subrayada.
  • <em></em>: Pone énfasis o recalca.
  • <strong></strong>: Parte importante.
  • <small></small>: Letra pequeña.
  • <s></s>: Contenido irrelevante u obsoleto.

Además existen dos etiquetas de salto de línea:

  • <br>: Añade una salto de línea.
  • <wbr>: Asigna una oportunidad de salto de línea. En el caso de que la frase tenga que pasar a la siguiente línea se dividirá en este punto.

Hiperenlace

Un hiperenlace, hipervínculo, o vínculo, es un enlace, que al seleccionarse lleva de una página web o archivo. Se usa le etiqueta <a></a> (anchor) y el atributo href="" para indicar la dirección. Por ejemplo: 

Enlace a google: <a href="https://www.google.es/">Google</a>

Se muestras en el navegador así: 

Enlace a google: Google

Podemos usar enlaces absolutos (https://www.google.es/) o relativos (pagina.html).
Otro uso de la etiqueta <a></a> (anchor) junto al atributo id="" nos permite crear un punto en esta página u otra al que acceder. Por ejemplo: 

Al comienzo de la página está escrito lo siguiente:
<a id="principio"></a>
Para ir al principio de la página creamos una enlace:
<a href="principio">Subir</a>
Este enlace se muestra de la siguiente forma en la página web:
Subir

También podemos hacer un enlace a un elemento. Debemos añadir el atributo id="" en el elemento al que queremos hacer referencia. Para crear el enlace escribimos lo siguiente: 

<a href="#nombre_id"></a>

En el caso de querer crear un enlace a un elemento de otra página web, debemos escribir lo siguiente: 

<a href="URL#nombre_id"></a>

Si quieres que se descargue el fichero al que hacer referencia el enlace, debemos añadir el atributo download. Por ejemplo:

<a href="Mifoto.png" download>Descargar foto</a>

Contenido encajado: imágenes

Para añadir una imagen a nuestra página web, debemos usar la etiqueta <img>. También debemos usar ciertos atributos:

  • src="": Lugar en el que se encuentra la imagen que queremos mostrar.
  • width="": Anchura de la imagen (Pixeles).
  • height="": Altura de la imagen (Pixeles).
  • alt="": Comentario descriptivo de la imagen.

Por ejemplo: 

<img src="perezoso.jpg" width="500" height="400" alt="Perezoso en una rama">

Mapas de imagen

Se pueden delimitar zonas de una imagen para que te lleven a un enlace. Para ello se usa la etiqueta <map></map> y la etiqueta <area> para definir cada zona de la imagen. Además son necesarios los atributos que se mencionan a continuación:

  • shape="": Es el tipo de área que queremos crear (default, rect, circle, poly).
  • coords="": Son las coordenadas para el tamaño del área.
  • href="": Determina la dirección a la que hacemos referencia con la zona.
  • alt="": Comentario descriptivo de la zona (Obligatorio si usamos "href").
  • download: Descarga el dichero del enlace.
  • tarjet="": Permite elegir como queremos que se abra el enlace.

Por ejemplo, vamos a crear una zona en la cabeza del perezoso que nos lleve a la página http://www.w3c.es: 

<img src="perezoso.jpg" alt="Perezoso en una rama" usemap="#w3c">
<map name="w3c">
	<area shape="circle" coords="325,325,35" href="http://www.w3c.es" alt="Pagina de w3c">
</map>

Tablas

Se pueden crear tablas con la etiqueta <table></table>. Dentro de la etiqueta usamos las siguientes etiquetas:

  • <tr></tr>: Crea una nueva fila a la que añadiremos columnas.
  • <td></td>: Crea las columna que necesitemos dentro de una fila.
  • <th></th>: Crea la columna de encabezado.
    • rowspan="": Es un atributo de <th> y <td>, ocupa el número de columnas hacia la derecha que se especifique.
    • colspan="": Es un atributo de <th> y <td>, ocupa el número de filas hacia abajo que se especifique.

Además <table></table> tiene el atributo border="" para añadir el tamaño del borde de la tabla.

			<th>10:00-11:00</th>

			<td>Trabajo</td>
			<td>Trabajo</td>
			<td>Trabajo</td>
